Creating Dataset from Data Source

For creating dataset, geographical data provided by http://www.geonames.org/ is used. The project for creating the dataset is located at Learn from Map / Data Set / sqlite3 / . This project is a netbeans project. There are few prior requirements before the project is runnable.

Requirements

The following files must be downloaded and placed in the project folder.

http://download.geonames.org/export/dump/allCountries.zip
http://download.geonames.org/export/dump/admin2Codes.txt
http://download.geonames.org/export/dump/admin1CodesASCII.txt

On sucessfull running the project, a new database file will be created containing clean and knowledge data that will be required for the Learn From Map app.

You can view the content of the database file using SQLite Database Browser

XML Question Rules

Question rules are stored in an XML file. All the rules are unders the parent questions. Each question rule is within question tag and consist of various eleemnts.

 <question>
<code>PPLC</code>
<type>FILL</type>
<alias>Capital</alias>
<marker></marker>
<relationship>country</relationship>
<location>true</location>
<format>Capital of :X: is _________</format>
<count>unique</count>
<answer>name</answer>
</question>

Each element is explained here

  1. Code: code represents the category of questions
  2. Type: type represents the type of question. Possible values : FILL, PIN, MCQ
  3. Alias: Alias name is used to store the category name of the question
  4. Marker: marker stores the column name. The value stored in the column name is displayed when the correct answer is shown.
  5. Relationship: It is used in the question generation logic.
  6. Format: The question format. :X: replaces the value stored in relationship column
  7. Count: unique means the rule will be used only once in a particular challenge
  8. Answer: stores the answer stored in column name specified.
